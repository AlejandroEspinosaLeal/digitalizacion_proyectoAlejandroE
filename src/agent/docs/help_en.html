File Sorter Enterprise - User Guide

Welcome to your intelligent file governance dashboard! Here is everything you need to know to automate your massive file storage organization.

1. Getting Started: The Home Page

  1. Select Source Folder: Drag & Drop any chaotic folder (like your Downloads) directly into the main dropzone, or click inside the container to browse for it manually.
  2. Toggle Filters: Safely uncheck any file categories you do not want the system to process (e.g., sort out the generic Documents but leave massive Video renders exactly where they stand).
  3. Start Organization: Click the primary green 'Start Organization' button. The intelligent engine will immediately categorize and move the files into highly structured new subdirectories mapped by logic paradigms.

2. Advanced Settings

  • Dry-Run Mode: Switch this to test your rules securely. The app will logically simulate the mapping workflow and output a status log telling you what is going to happen, but without actually modifying any bytes inside your physical drives.
  • Continuous Watchdog: Enable this daemon to run perpetually and silently in the background alongside the OS. It will automatically detect, scan, and sort newly downloaded items within fractions of seconds upon them hitting the watched route.
  • Undo Batch: Made a fundamental mistake during deployment? No panic. Gently click the red ⏪ Undo Batch to trigger a guaranteed 100% mechanical rollback taking all swept assets entirely back to their primary origins reliably.
  • Exclusions: Type comma-separated string keywords (e.g., draft, temp, private) in the ignore-list input to instruct the crawler to entirely blindside files matching the syntax structure.

3. Metrics Dashboard

Switch gears to the Metrics Dashboard endpoint found natively on the left navigation strip to overview an encompassing breakdown of all the items safely rescued up. Monitor high-level graphs defining success metrics based uniquely structured against hierarchical groups in real-time context.

4. Automata & Local Rules

Demanding enterprise categorization? Move downwards to the Rules Engine view logic. You can enforce granular overrides allowing you to intercept precise string identifiers matching defined extensions directly bypassing generic folders altogether.

  • Algorithm Example: If mapped Extension happens to be .pdf + AND String parameter holds invoice 🡆 Omit default handling and redirect routing to targeted absolute Finance Logs custom bin.
