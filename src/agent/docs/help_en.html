User Guide - Enterprise File Governance

Welcome! This tool organizes your storage instances automatically based on state-of-the-art hierarchical directory tracking.

1. Source Directory Input

Click on the main central hub to explore for an unstructured folder. NEW: You can also seamlessly Drag & Drop target folders directly into the UI dropzone.

2. Premium Productivity Engine

  • Rollback Engine (Undo): The red ⏪ Deshacer Lote action button allows a full 1oo% guaranteed rollback of the latest sweep iteration, returning millions of moved files exactly back to their origin path safely.
  • Analytics Dashboard: Using an offline persistent JSON memory state, the Dashboard tab displays live historical aggregated success rates and file statistics.
  • Custom Local Rule Engine: Build your own algorithmic routing mappings assigning precise text-keyword matches and strict file extensions to your own custom directory folders.

3. Core Parameter Toggles

  • Dry-Run Mode (Simulacro): Safely evaluates pattern-matches and simulates the internal target paths WITHOUT moving any real bytes across disks. Ideal for testing custom rules.
  • Recursive Scanning (Subcarpetas): Drops into endless labyrinth sub-directories discovering and gathering stray files to centralize them via the high-performance 'os.scandir' iterator.
  • Date Sorting Hook: Wraps categorized outputs inside auto-generated Year/Month directory nodes bypassing flat congestion.
  • Continuous Watchdog Hook: Silent daemon that runs perpetually in the background sorting inbound payloads instantly.
  • Exclusion Overrides: Type comma-separated keywords like draft, temp to strictly ignore targeted items.

4. Batch Execution Layout

Running the main process spawns threaded unblocking background executors. Upon completion, a comprehensive HTML Security Audit log is SMTP-delivered seamlessly toward your registered enterprise email inbox.
